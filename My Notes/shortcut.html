Shortcut Description
Ctrl+Enter Create new line without getting to end of current line.
Alt+(Up OR Down) Arrow Move a line up & down
Ctrl+/ Start a comment
Ctrl+(Left OR Right) Arrow Move cursor word by word rather than character by character
Ctrl+(Up OR Down) Arrow Scroll up or down
Ctrl+Alt+Shift+(2 OR 8) Replicate the current line to bottom
Alt+Click Multiple cursors


Emmet Shortcuts Description
! Boilerplate code
Type Tag Name without '<' Auto-complete the tag
lorem Sample text
loremN (N is a number) N number of sample words
tagname*N (N is a number) Repeat a tag N times
